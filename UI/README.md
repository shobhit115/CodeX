# CodeX UI & Design Assets

This directory contains user interface mockups, design assets, and graphical resources used throughout the CodeX platform.

## Contents

Currently, you will find the following visual assets:

- **`landing-1.jpeg`**, **`landing-2.jpeg`**, **`landing-3.jpeg`**: High-fidelity mockups or screenshots representing the layout and design of the CodeX platform's landing pages. These are used as references by the Frontend team to ensure the implemented UI matches the design specifications.

## Usage

When building new features on the frontend:
1. Refer to these designs to ensure visual consistency in layout, colors, and typography.
2. If new designs are created (e.g., via Figma or Adobe XD), please export high-quality screenshots and store them here for historical reference.
3. Actual application assets (like logos, icons, etc.) that need to be served to the browser should be placed in `Frontend/public` or `Frontend/src/assets`, not here. This folder acts as an archive and reference for designers and developers.
